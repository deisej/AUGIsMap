AUGIsMap — Mapping Geographically Isolated Wetlands (Brazil)

Este repositório reúne os scripts do Google Earth Engine utilizados no doutorado de Deise Junqueira para o inventário, mapeamento e diagnóstico ambiental das Áreas Úmidas Geograficamente Isoladas (AUGIs) na Depressão Periférica Paulista, com foco inicial na Bacia Hidrográfica do Rio Corumbataí.

O projeto integra fotointerpretação manual, classificação supervisionada por Random Forest, embeddings espectrais, variáveis geomorfológicas e filtros morfométricos pós-classificação para detectar e caracterizar wetlands isoladas da rede de drenagem.
